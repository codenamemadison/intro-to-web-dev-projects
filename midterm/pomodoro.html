The Pomodoro Technique

About

The Pomodoro Technique is a time management method created by Francesco Cirillo during the late 1980's. It is a system based on working into intervals, separated by short breaks. This allow users who typically get distracted to work for a set amount of time, knowing they also have a set time to be "distracted". The technique is great for students, procrasinators, and anyone else who has a hard time focusing. The technique has been widely popularized by numerous websites and apps.

How does it work?

  1. Choose the task that must be accomplished

  2. Set a timer for a certain amount of time to do work (traditionally 25 minutes)

  3. Work on said task

  4. Once the timer is over, take a short break (typically from 5-10 minutes)

  5. Repeat Steps 2-4 until you have done three rounds (or three "pomodoros")

  6. After three pomodoros, do a fourth pomodoro but instead of a short break take a long break (typically 20-30 minutes).

  7. Start from Step #2 and repeat the cycle until the task is done
